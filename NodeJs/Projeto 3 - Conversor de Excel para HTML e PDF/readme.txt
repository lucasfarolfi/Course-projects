# Bibliotecas usadas #

*fs - Para manipulação de arquivos - Já é do node
*util - Para utilizar o metodo .promisify para converter um callback para promise - Já é do node
*ejs - Gerar um html dinâmico com base nos dados da tabela do excel - npm install ejs --save
*html-pdf - Converte os dados html para um arquivo PDF - npm install html-pdf --save
